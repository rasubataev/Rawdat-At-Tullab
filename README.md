# Rawdat At-Tullab — платформа для изучения арабского языка

Мобильное PWA-приложение для системного изучения арабского языка на основе шариатских текстов. Не словарь и не разговорник — полноценная учебная система с интервальным повторением, структурированной библиотекой книг и интерактивными уроками грамматики.

---

## Продукт

Rawdat At-Tullab строится вокруг одной идеи: студент шариатских наук должен изучать арабский язык через оригинальные тексты, а не через переводные учебники. Приложение объединяет словарный запас из программных книг (бейна ядейк, аджрумия, таджвид, фикх) с алгоритмом интервального повторения — так слова запоминаются в контексте, а не в вакууме.

Текущая база: **3304 слова** из «Аль-Арабия бейна ядейк» (все 8 частей, 64 вахьдата). В разработке: Аджрумия, Таджвид аль-Мусаввир, Матн Гаят ат-Такриб. Целевой объём к релизу — **10 000+ лексических единиц**.

---

## Ключевые возможности

**Интервальное повторение (SM-2)**
Адаптированный алгоритм SM-2 с тремя уровнями оценки (трудно / нормально / легко). Карточка возвращается через 10 минут, 1 день, 3 дня или более — в зависимости от оценки и коэффициента лёгкости. Прогресс сохраняется в localStorage и экспортируется в JSON.

**Структурированная библиотека**
Книги → части → вахьдаты → слова. Каждый уровень иерархии отображает прогресс, процент изученного и доступность. Новые книги подключаются без изменения архитектуры.

**Интерактивные уроки**
Формат урока: арабский текст правила → объяснение → категориальные карточки → секции с признаками и примерами → итоговый совет. Реализовано для грамматики (Аджрумия) и таджвида.

**Персональные колоды**
Пользователь создаёт собственные наборы слов, добавляет через форму или массовым импортом (формат `ar ; ru`). Колоды участвуют в смешанных сессиях.

**Статистика и серии**
Тепловая карта активности за 91 день, счётчик серий, точность за 7 дней, динамика изученных слов.

## Архитектура и технологии

```
Rawdat At-Tullab
├── index.html          — единственный HTML-файл, все экраны через display:none
├── app.js              — вся логика: SRS, навигация, рендеринг, контент (~5000 строк)
├── style.css           — дизайн-система: токены, компоненты, темы, адаптив
├── manifest.json       — PWA манифест
└── icons/              — обложки книг, иконка приложения
```

**Стек:** Vanilla JS (ES6+), CSS Custom Properties, LocalStorage API, PWA (Service Worker ready).

**Шрифты:** Tajawal (интерфейс и главный экран), SF Arabic / Noto Naskh Arabic (карточки и уроки — классический каллиграфический стиль).

**Темы:** светлая и тёмная через `data-theme` атрибут и CSS-переменные. Переключение без перезагрузки.

**Адаптив:** телефон (основной), планшет (боковая навигация, 3 колонки), ноутбук (4 колонки). Breakpoints: 768px, 1100px.

## Принцип работы

Каждое арабское слово идентифицируется хешем от арабской строки (`keyOf(ar)`). Карточка хранит: `ease` (коэффициент лёгкости), `interval` (дней до следующего показа), `due` (timestamp), `reps`, `lapses`, флаги (`hard`, `fav`, `learned`).

При оценке `hard`: интервал сбрасывается, карточка возвращается через 10 минут, ease уменьшается на 0.20. При `ok`: интервал растёт по формуле `interval × ease`. При `easy`: интервал растёт на 30% быстрее, ease увеличивается на 0.10.

Навигация — через кастомный роутер без фреймворков: `navigate(screenId)` скрывает все экраны и показывает нужный, стек истории хранится в `NAV_STACK[]`.

Весь контент (слова, уроки, метаданные книг) находится в `BOOKS[]` и `LESSONS{}` в `app.js`. Рендеринг — шаблонные строки JS, делегирование событий через один `document.addEventListener('click')`.

---

## Масштабируемость

**База слов** расширяется добавлением объектов в `BOOKS[]`. Новая книга подключается за ~10 строк кода — структура `{ id, ar, title, cover, parts[] }`. Слова хранятся в вахьдатах и автоматически попадают в SRS, поиск, статистику и смешанные сессии.

**Уроки** добавляются в объект `LESSONS{}` с ключом по `part.id`. Формат урока декларативный — массивы секций, карточек, примеров. Рендерер `renderLesson()` обрабатывает любую структуру.

**Целевой объём:** 50 000+ слов при подключении полных корпусов программных книг шариатских факультетов.
